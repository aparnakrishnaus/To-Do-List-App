## To-Do List App - Task Management Web App

A responsive to-do list application designed to help users manage daily tasks efficiently while staying motivated with inspirational quotes and theme customization.

## Live Preview
 https://todobuddylist.netlify.app/

## GitHub Repository
 https://github.com/aparnakrishnaus/To-Do-List-App

## Features
- Create, edit, and delete daily tasks
- Minimal and distraction-free task management UI
- Displays motivational quotes to boost productivity
- Light and dark theme support
- Fully responsive design for all devices

## Technologies Used
- HTML5
- CSS3
- JavaScript
